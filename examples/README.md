# Examples

Documented harness runs against real target repositories.

## v0.1 status

**No examples yet.** This directory will hold anonymized or redacted artifacts from manual v0.1 loops once they are run.

## First planned example

**Target repo:** [`weston-uribe/weston-uribe-portfolio`](https://github.com/weston-uribe/weston-uribe-portfolio)

**Expected artifacts per run:**

- Filled `linear-issue.md` (or equivalent issue export)
- `implementation-plan.md`
- `eval-scorecard.md`
- `pr-readiness-report.md`
- Short retrospective in `docs/research/`

## What to include in an example

- Issue summary and scope
- What the agent changed (high level)
- Eval results and open questions
- What would change in templates or rubrics next time

## What not to include

- Secrets, tokens, or private URLs
- Unreviewed code dumps
- Claims that the run was fully automated
